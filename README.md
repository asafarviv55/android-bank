# Android Bank

Android banking application with Navigation component.

## Tech Stack

- Java
- Android SDK
- Navigation Component
- Material Design

## Prerequisites

- Android Studio Arctic Fox or later
- JDK 11+
- Android SDK 31+

## Setup

1. Open in Android Studio
2. Sync Gradle
3. Run on emulator or device

## Build

```bash
./gradlew assembleDebug
```

## Project Structure

```
app/src/main/
├── java/com/alfa/bank/
│   ├── MainActivity.java
│   ├── FirstFragment.java
│   └── SecondFragment.java
├── res/
│   ├── layout/
│   └── navigation/nav_graph.xml
└── AndroidManifest.xml
```
